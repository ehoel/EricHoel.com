Weclome to erichoel.com!  Thanks for stopping by.  A little about me:

I enjoy learning new technologies and applying that knowledge to problems.  I graduated college with a mircobilogy degree.  Since then, I've lerned SQL, VBA, Unix, some HTML and JavaScript.  I discovered my passion lied in data and was fortunate enough to get the opportunity to learn SQL.  In my free time I've successfully set up a home media server using ubuntu server 14.04 LTS.  The server backs up my files, hosts my media, and servers as a media player with a web interface.  I created a temperature contolled fermentation vessle using raspberry pi, an adruino switch and the brewpi software.  This keeps my beer fermenting at the proper temperature wtih a web interface that graphs the temperature.

 

I got my start in databases by learning about the VLOOKUP function in excel.  I consolidated numerous web and internal sources into an workbook.  VLOOKUPs pulle the URL or data.  This created a single searchable source and saved a lot of time.

From there I ventrue into the world of SQL and relational databases.  I taught myself SQL and have acomplished many projects dealing with reporting, ETL, quality assurane and programing.  I started this website to challenge myself with working with a webapp connected to a database.
